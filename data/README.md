# Data

This project uses data collected from YouTube videos related to the Brexit referendum.

## Data Source

The data were collected using the YouTube Data API from videos published on the BBC News YouTube channel.

The analysis focused on videos associated with the Brexit referendum and discussions about immigration in the United Kingdom.

## Data Collected

The collection process included:

- Video metadata
- Video descriptions
- Publication dates
- User comments
- Comment metadata (likes and timestamps)

## Data Processing

The raw comments were cleaned and preprocessed before analysis.

Additional variables were generated, including:

- Immigration semantic similarity scores
- Toxicity scores obtained with Detoxify

## Reproducibility

Researchers interested in reproducing the analysis can obtain their own YouTube Data API credentials and follow the data collection workflow implemented in the notebook provided in this repository.

## Notes

The original dataset is not distributed with this repository.

Data were collected exclusively for academic and educational purposes within a Computational Social Science project focused on immigration discourse during the Brexit referendum.
